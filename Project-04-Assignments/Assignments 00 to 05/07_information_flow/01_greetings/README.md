# 🎉 **01_Greetings**  

🔹 This program **greets the user** with a personalized message.  

#### ⚡ **How It Works?**  
1️⃣ User enters their **name**.  
2️⃣ The `greet()` function **formats the greeting message**.  
3️⃣ It **returns** a greeting like `"Greetings Zara!"`.  

#### 📝 **Example Output:**  
```
What's your name? Zara
Greetings Zara!
```
